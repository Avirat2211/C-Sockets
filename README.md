## Prerequisites
Ensure you have a c-compiler
 
---

## Running 
1. To start server, navigate to /server then
```bash
gcc main.c ../Socketutil/src/socketutil.c -o a.out
```
```bash
./a.out
```



2. To start clientside, navigate to /client then

```bash
gcc main.c ../Socketutil/src/socketutil.c -o a.out
```

```bash
./a.out
```
